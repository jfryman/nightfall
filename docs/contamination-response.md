# Contamination Response

If forbidden-source provenance is suspected:

1. Stop work on the affected subsystem immediately.
2. Record the incident in `docs/contamination-log.md`.
3. Open a blocker doc describing the affected area and why provenance is in
   doubt.
4. Remove or quarantine suspect work only after the blocker state is captured.
5. Resume only after explicit adjudication with `**Resume:** yes`.

This file is bootstrap guidance based on the clean-room rules summarized in
`AGENTS.md` and the decisions log.
